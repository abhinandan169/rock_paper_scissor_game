# Rock Paper Scissors Game

A simple Rock Paper Scissors game built using Python. The player competes against the computer, and the winner is decided based on the game rules.

## Features

- Play Rock, Paper, Scissors against the computer
- Random computer choice
- User input validation
- Win, Lose, or Tie result
- Play multiple rounds using a loop

## Technologies Used

- Python 3
- random module

## Project Structure

```
rock_paper_scissor_game/
│── main.py
│── README.md
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/abhinandan169/rock_paper_scissor_game.git
```

2. Go to the project folder

```bash
cd rock_paper_scissor_game
```

3. Run the program

```bash
python main.py
```

## Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
